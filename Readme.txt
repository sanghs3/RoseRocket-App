# RoseRocket-App

Instructions:

Server-side: 
1. Open cmd and navigate to roserocketServer
2. Run: "npm install"
3. Open cmd: "npm start"

Client-side: 
1.Open second cmd and navigate to roserocketweb
2. Run: "npm install"
3.Run command: "npm appStart"
4.Go to localhost:3001
5.Click Start button
6. Enjoy!
