# RoseRocket-App

Instructions:<br />

Server-side: <br/>
1. Open cmd and navigate to roserocketServer
2.Open cmd: "npm start"<br />

Client-side: <br/>
1.Open second cmd and navigate to roserocketweb
2.Run command: "npm appStart"<br />
3.Go to localhost:3001<br />
4.Click Start button<br />
5. Enjoy!<br />
